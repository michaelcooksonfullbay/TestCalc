Calculator Memory

Small, self-contained feature — four buttons:

  • MC — clear stored value
  • MR — recall stored value
  • M+ — add display to memory
  • M− — subtract display from memory

An M indicator appears when a value is stored.

API

Two new endpoints: GET and PUT on /memory.

Scope

This feature is independent of existing calculator operations. No changes to arithmetic, history, or authentication. Existing tests should not be affected.
