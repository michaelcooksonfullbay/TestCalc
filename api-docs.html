TestCalc API

v1.0.0

TestCalc REST API

Backend API for the TestCalc calculator application. Provides authentication, calculation, and history management endpoints.

Base URL: https://api.testcalc.local/v1
Authentication
POST /api/auth/login Authenticate user ▼

Request Body

{
  "username": "alice",
  "password": "password123"
}

Responses

200 Successful login 
{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "user": {
    "id": "usr_a1b2c3",
    "username": "alice",
    "createdAt": "2024-01-15T08:30:00Z"
  }
}
401 Invalid credentials 
{
  "error": "INVALID_CREDENTIALS",
  "message": "Invalid username or password"
}
POST /api/auth/logout End session ▼

Parameters

NameLocationTypeRequired
Authorization header string required

Responses

200 Successfully logged out 
{
  "message": "Session ended"
}
401 Not authenticated 
{
  "error": "UNAUTHORIZED",
  "message": "No valid session"
}
History
GET /api/users/{userId}/history Get calculation history ▼

Parameters

NameLocationTypeRequired
userId path string required
Authorization header string required
limit query integer optional
offset query integer optional

Responses

200 History entries 
{
  "userId": "usr_a1b2c3",
  "entries": [
    {
      "id": "calc_x7y8z9",
      "expression": "5 + 3",
      "result": "8",
      "timestamp": "2024-06-10T14:22:00Z"

    {
      "id": "calc_m4n5o6",

Calculator
POST /api/calculate Perform calculation ▼

Parameters

NameLocationTypeRequired
Authorization header string optional

Request Body

{
  "expression": "5 + 3 * 2"
}

Responses

200 Calculation result 
{
  "expression": "5 + 3 * 2",
  "result": "11",
  "precision": "float64",
  "savedToHistory": true
}
400 Invalid expression 
{
  "error": "INVALID_EXPRESSION",
  "message": "Malformed expression: unmatched parenthesis"
}
GET /api/calculate/validate Validate expression ▼

Parameters

NameLocationTypeRequired
expression query string required

Responses

200 Validation result 
{
  "expression": "5 + 3",
  "valid": true,
  "tokens": ["5", "+", "3"],
  "warnings": []
}
200 Invalid expression 
{
  "expression": "5 + + 3",
  "valid": false,
  "tokens": null,
  "warnings": ["Consecutive operators at position 4"]
}
400 Missing expression parameter 
{
  "error": "MISSING_PARAMETER",
  "message": "Query parameter 'expression' is required"
}
